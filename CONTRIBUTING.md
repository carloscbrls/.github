# Contributing to CC3PO

Thank you for your interest in contributing! This is a private ecosystem managed by Atlas 🌍.

## How to Contribute

1. **Fork the repository** you want to work on
2. **Create a feature branch** (`git checkout -b feature/your-feature`)
3. **Make your changes** following the project's code style
4. **Commit with clear messages** using conventional commits
5. **Push and open a Pull Request**

## Code Style

- Use TypeScript for all new code
- Follow existing patterns in the codebase
- Keep functions small and focused
- Add comments for non-obvious logic

## Pull Request Process

1. Ensure your code builds (`npm run build`)
2. Update documentation if needed
3. Link any related issues
4. Request review from the maintainer

## Questions?

Open a discussion or contact legal@cc3po.com
