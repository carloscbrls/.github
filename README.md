# 🏛️ CC3PO — Default Community Health Files

> This repo provides **default community health files** for all 59 CC3PO GitHub repositories.
> Every repo inherits these automatically — no need to copy them.

## What's Here

| File | Purpose |
|------|---------|
| `CONTRIBUTING.md` | How to contribute to CC3PO projects |
| `SECURITY.md` | How to report security vulnerabilities |
| `SUPPORT.md` | Where to get help |
| `CODE_OF_CONDUCT.md` | Community standards |
| `LICENSE` | MIT License (all repos) |
| `REPO-CATALOG.md` | **Full catalog of all 59 repos** — for partners |
| `.github/dependabot.yml` | Auto dependency updates (weekly) |
| `.github/workflows/dependabot-auto-merge.yml` | Auto-approve & merge low-risk dep PRs |

## For Partners

See **[REPO-CATALOG.md](./REPO-CATALOG.md)** for the complete map of every CC3PO repository.

## How Defaults Work

GitHub automatically applies these files to any repo in the `carloscbrls` org that doesn't have its own version. To override for a specific repo, create the file in that repo's root.
